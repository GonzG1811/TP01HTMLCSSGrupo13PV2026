# TP01 HTML/CSS GRUPO 13 - PROGRAMACION VISUAL
Este repositorio contiene el codigo fuente de un Gestor de proyectos educativos. El proyecto esta compuesto por las carpetas:

assets/: Carpeta principal.

    css/: Contiene las hojas de estilo para el diseño visual.

    img/: Almacena todas las imágenes y recursos multimedia.

    pages/: Contiene las diferentes secciones y páginas secundarias del sitio:

        index.html: La página principal para entrar al sitio de gestion.

        Detalle-cpp.html: Detalles sobre el proyecto C++.

        detalle-js.html: Detalles sobre el proyecto JavaScript.

        Detalle.html: Página general de detalles.

        perfil.html: Apartado sobre el perfil del usuario.

        Proyectos.html: Galería de proyectos realizados.
